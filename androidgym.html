Descrição do projeto

Desenvolvimento de uma aplicação Android para gestão de vendas, integrada com uma API RESTful. A aplicação oferece funcionalidades como compra de produtos, carrinho de compras, geração de faturas e visualização de favoritas offline.

Técnologias Utilizadas

  • Java
  • Android Studio
  • SQLITE
