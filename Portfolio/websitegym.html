Descrição do projeto

Estou a desenvolver um website para a gestão de vendas de produtos de ginásio. O projeto incluiu um front-office para interação com os clientes e um back-office para administração interna, com autenticação segura e controle de acesso baseado em roles (RBAC).

Técnologias Utilizadas

  • PHP
  • Framework Yii2
  • MySQL
  • MAMP-Ambiente de desenvolvimento local
